# jsrc Documentation (EN)

## Quick Start

```bash
git clone https://github.com/imjiaoyuan/jsrc.git
cd jsrc
uv venv
uv sync --extra dev
uv run jsrc --help
```

Or:

```bash
pip install -e .
jsrc --help
```

Run format:

```bash
jsrc <module> <subcommand> [options]
```
